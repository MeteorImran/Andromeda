## 🇬🇧 English

> **Sovereign Node Orchestration** — A cross-platform, local-first AI assistant with enterprise-grade orchestration capabilities and a deep-space telemetry HUD.

### Overview
ANDROMEDA is a fully local, real-time voice and visual AI agent designed for privacy, speed, and autonomous task execution. It removes cloud dependencies by leveraging locally hosted LLMs (via Ollama or OpenAI-compatible APIs), offline speech recognition, and high-fidelity text-to-speech.

### Key Features
- **Galaxy Engine HUD** — A dynamic UI with starfield animations and neural constellation maps.
- **Live Telemetry** — Real-time monitoring of CPU, Memory, Network, GPU, and Temperature.
- **Zero-Trust Sandbox** — Secure file dropzone for processing images, PDFs, CSVs, and more.
- **Streaming Orchestration** — Voice synthesis begins at the first generated sentence.
- **Autonomous Tool Calling** — 20 built-in tools for browser automation, file management, and system control.

### Detailed Architecture
1. **UI Layer (Galaxy Engine)**: Built with PyQt6, featuring a custom canvas for high-performance starfield animations and real-time telemetry monitoring (CPU, Mem, Net, GPU, Temp) using `psutil`.
2. **Acoustic Sensor (STT)**: Utilizes a Voice Activity Detection (VAD) buffer to capture audio, which is then processed by `faster-whisper` or `Vosk` for offline speech-to-text.
3. **Cognitive Engine (LLM)**: Interfaces with Ollama or OpenAI-compatible APIs. Features a "Warmup" sequence that primes the KV prefix cache with the system prompt to reduce first-token latency to <1s.
4. **Tool Orchestration**: A multi-step autonomous task queue that can chain 20+ built-in tools (Browser, Files, OS) to solve complex goals with error recovery.
5. **Voice Synthesis (TTS)**: Employs a streaming worker that begins speech synthesis (via EdgeTTS, Kokoro, or ElevenLabs) as soon as the first sentence is generated by the LLM.

### Quick Start
```bash
# 1. Install Ollama and pull model
ollama pull qwen2.5:7b

# 2. Launch
cd Mark-XL-main
python main.py
```

## 🇲🇾 Bahasa Melayu

> **Orkestrasi Nod Berdaulat** — Pembantu AI tempatan silang-platform dengan keupayaan orkestrasi gred perusahaan dan HUD telemetri angkasa lepas.

### Ringkasan
ANDROMEDA ialah ejen AI suara dan visual masa nyata tempatan sepenuhnya yang direka untuk privasi, kelajuan, dan pelaksanaan tugas autonomi. Ia menghapuskan kebergantungan awan dengan menggunakan LLM hos tempatan (melalui Ollama atau API serasi OpenAI), pengecaman pertuturan luar talian, dan sintesis teks-ke-suara kesetiaan tinggi.

### Ciri-ciri Utama
- **HUD Galaxy Engine** — Antara muka dinamik dengan animasi medan bintang dan peta buruj neural.
- **Telemetri Langsung** — Pemantauan masa nyata untuk CPU, Memori, Rangkaian, GPU, dan Suhu.
- **Sandbox Zero-Trust** — Zon drop fail selamat untuk memproses imej, PDF, CSV, dan lain-lain.
- **Orkestrasi Penstriman** — Sintesis suara bermula pada ayat pertama yang dijana.
- **Panggilan Alatan Autonomi** — 20 alatan terbina untuk automasi pelayar, pengurusan fail, dan kawalan sistem.

### Seni Bina Terperinci
1. **Lapisan UI (Galaxy Engine)**: Dibina dengan PyQt6, menampilkan kanvas tersuai untuk animasi medan bintang berprestasi tinggi dan pemantauan telemetri masa nyata (CPU, Mem, Net, GPU, Suhu) menggunakan `psutil`.
2. **Sensor Akustik (STT)**: Menggunakan penimbal Pengecaman Aktiviti Suara (VAD) untuk menangkap audio, yang kemudiannya diproses oleh `faster-whisper` atau `Vosk` untuk penukaran suara-ke-teks luar talian.
3. **Enjin Kognitif (LLM)**: Berinteraksi dengan API Ollama atau serasi OpenAI. Mempunyai urutan "Pemanasan" yang menyediakan cache awalan KV dengan arahan sistem untuk mengurangkan kependaman token pertama kepada <1s.
4. **Orkestrasi Alatan**: Barisan tugas autonomi berbilang langkah yang boleh merantaikan 20+ alatan terbina (Pelayar, Fail, OS) untuk menyelesaikan matlamat kompleks dengan pemulihan ralat.
5. **Sintesis Suara (TTS)**: Menggunakan pekerja penstriman yang memulakan sintesis pertuturan (melalui EdgeTTS, Kokoro, atau ElevenLabs) sebaik sahaja ayat pertama dijana oleh LLM.

### Permulaan Pantas

```bash
ollama pull qwen2.5:7b

# 2. Lancarkan
cd Mark-XL-main
python main.py
```
